
자바의 열거형(enum) 을 대체할 수 있는 사용자 정의 클래스를 작성하세요.

1. 클래스의 기능적 요구사항

   아래의 각 요구사항 항목에는 실제 enum 의 실 사용 코드가 있습니다.
   구현된 JAVA 클래스가 enum 의 실 사용 코드와 최대한 비슷하게 동작할 수 있도록
   JAVA 언어로 구현하세요.

   1.1 기본적인 enum 형태로 사용할 수 있어야 함.
		enum Direction { EAST, SOUTH, WEST, NORTH };

   1.2 enum을 대체할 클래스에 각 상수별 교통비 계산의 기능적인 요소가 포함되도록 해야 함.
		enum Transportation { 
			BUS(100)      { int fare(int distance) { return distance*BASIC_FARE;}},
			TRAIN(150)    { int fare(int distance) { return distance*BASIC_FARE;}}
		};

2. 구현 기능의 동작 확인
   각 기능은 구현된 메소드를 사용해서 동작 확인이 되도록 하세요.

   2.1 상수 참조 변수 선언
	Transportation t1 = Transportation.BUS;

   2.2 상수명, 상수의 내부 관리 정수값 출력
	t1.name() 메소드, t1.ordinal() 메소드를 사용해서 출력

   2.3 Transportation.BUS.fare(100)
	Transportation.BUS.fare(100) 메소드를 활용해서 출력
